# 画像フォルダ

このフォルダには、LPで使用する画像を配置してください。

## 必要な画像ファイル

### ヘッダー用画像
- **ファイル名**: `header-snow.jpg` (または `.png`, `.webp`)
- **推奨サイズ**: 幅 1200px 以上、高さ 400px 以上
- **用途**: ヘッダー背景に使用される北海道の雪景色の画像
- **推奨内容**: 
  - 札幌の雪景色
  - 小樽の雪景色
  - 北海道の冬の風景
  - 上品で落ち着いた雰囲気の画像

### フッター用画像（オプション）
- **ファイル名**: `footer-snow.jpg` (または `.png`, `.webp`)
- **推奨サイズ**: 幅 1200px 以上、高さ 300px 以上
- **用途**: フッター背景に使用される画像（ヘッダーと同じ画像でも可）

## 画像の追加方法

1. このフォルダに画像ファイルを配置してください
2. ファイル名を上記の推奨名に変更してください
3. ブラウザで確認してください

## 注意事項

- 画像がない場合でも、グラデーション背景が表示されるため、サイトは正常に動作します
- 画像ファイルは適切なサイズに最適化することを推奨します（ファイルサイズを小さくする）
- 著作権に注意し、使用可能な画像を使用してください
