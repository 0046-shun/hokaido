# 結婚のご報告と両家顔合わせ LP 設計書

## 1. プロジェクト概要

### 1.1 目的
大城 駿様と新城 友梨様の婚約報告と、両家顔合わせ旅行のご案内を目的としたランディングページ（LP）の作成。

### 1.2 ターゲット
- 新城家の皆様
- 大城家の皆様
- 主にスマートフォンでの閲覧を想定

### 1.3 要件
- スマートフォンファーストのレスポンシブデザイン
- 読みやすく、分かりやすい情報構成
- 美しく上品なデザイン（結婚式らしい雰囲気）
- スムーズなユーザー体験

## 2. 技術仕様

### 2.1 使用技術
- HTML5
- CSS3（カスタムプロパティ、Flexbox、Grid、アニメーション）
- JavaScript（Vanilla JS）
- 外部ライブラリなし（軽量性重視）

### 2.2 ファイル構成
```
hokka/
├── index.html      # メインHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScriptファイル
└── DESIGN.md       # 本設計書
```

### 2.3 ブラウザ対応
- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- モバイルブラウザ（iOS Safari, Chrome Mobile）
- レスポンシブデザイン（320px以上）

## 3. デザイン仕様

### 3.1 カラーパレット
- **プライマリカラー**: `#667eea`（紫系グラデーションの開始色）
- **セカンダリカラー**: `#764ba2`（紫系グラデーションの終了色）
- **背景色**: `#f5f7fa` → `#c3cfe2`（グラデーション）
- **テキスト色**: `#333`（メイン）、`#555`（サブ）、`#666`（補助）
- **アクセント色**: `#ff6b6b`（重要情報のハイライト）

### 3.2 タイポグラフィ
- **フォントファミリー**: 
  - `'Hiragino Kaku Gothic ProN'`
  - `'Hiragino Sans'`
  - `'Yu Gothic'`
  - `'Meiryo'`
  - `sans-serif`（フォールバック）
- **フォントサイズ**:
  - 見出し1（h1）: 24px（モバイル: 20px/18px）
  - 見出し2（h2）: 22px（モバイル: 20px）
  - 見出し3（h3）: 18px
  - 本文: 15px
  - 小見出し: 14px

### 3.3 レイアウト
- **最大幅**: 800px
- **パディング**: 40px（デスクトップ）、30px（タブレット）、20px（モバイル）
- **マージン**: 自動（中央揃え）

## 4. コンポーネント設計

### 4.1 ヘッダー
- **背景**: 紫系グラデーション（`#667eea` → `#764ba2`）
- **アニメーション**: 背景にドットパターンの回転アニメーション
- **コンテンツ**: タイトルとサブタイトル

### 4.2 ナビゲーション
- **位置**: スティッキー（固定）
- **スタイル**: 白背景、紫のボーダー付きボタン
- **機能**: スムーズスクロール、現在セクションのハイライト

### 4.3 セクション
各セクションは以下の構造：
- セクションID（アンカーリンク用）
- 見出し（h2）
- コンテンツ（段落、テーブル、タイムラインなど）

### 4.4 スケジュールテーブル
- **スタイル**: グラデーション背景のヘッダー
- **レスポンシブ**: モバイルでは横スクロール可能

### 4.5 タイムライン
- **デザイン**: 縦線とドットで時系列を表現
- **カード形式**: 各日程をカードとして表示
- **色分け**: 時間は紫、場所はグレー

### 4.6 イベントカード
- **背景**: グレー系グラデーション
- **左ボーダー**: 紫のアクセント
- **レストラン名**: リンク付き（外部リンク）

### 4.7 ホテルカード
- **スタイル**: 白背景、グレーボーダー
- **リンク**: 公式サイトへの外部リンク

### 4.8 フッター
- **背景**: ヘッダーと同じグラデーション
- **コンテンツ**: 署名

## 5. インタラクション設計

### 5.1 スクロールアニメーション
- **フェードイン**: セクションが表示領域に入るとフェードイン
- **実装**: Intersection Observer API

### 5.2 ナビゲーション
- **スムーズスクロール**: アンカーリンククリック時にスムーズにスクロール
- **アクティブ状態**: 現在のセクションに対応するナビゲーションリンクをハイライト

### 5.3 ホバーエフェクト
- **ナビゲーションリンク**: ホバー時に背景色が変わる
- **外部リンク**: ホバー時に色が変わる

## 8. パフォーマンス最適化

### 8.1 軽量化
- 外部ライブラリ不使用
- インラインスタイル最小化
- CSS/JSの外部ファイル化（キャッシュ可能）

### 8.2 読み込み最適化
- CSSは`<head>`内で読み込み
- JavaScriptは`</body>`前で読み込み
- 画像は使用していないため、画像最適化は不要

## 9. アクセシビリティ

### 9.1 セマンティックHTML

### 9.2 キーボード操作

- フォーカス状態の視覚的フィードバック

### 9.3 スクリーンリーダー
